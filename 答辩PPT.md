- 可扩展 — 新交互模式只需添加新Agent

**意图分类**：LLM分类 + 关键词兜底，毫秒级路由

### 第7页 · 多模态教学详解（PPT/Word生成）

**AI一键生成教学课件**

输入一个主题 → LLM生成结构化内容 → 自动构建PPT/Word

**PPT生成流程**：
1. LLM生成6-8页幻灯片内容（JSON格式）
2. 构建Office Open XML（PPTX = ZIP + XML）
3. 输出可下载的.pptx文件

**Word生成流程**：
1. LLM生成结构化文档（目标→概念→讲解→例子→思考题）
2. 构建.docx文件
3. 输出可下载的.docx文件

**双轨实现**：优先python-pptx/python-docx，自动降级为纯Python构建
**后台上传**：支持PDF/DOC/PPT/TXT上传，自动提取文本并索引到知识库

### 第10页 · 学段自适应机制

**三层自适应架构**

```
第一层 · System Prompt注入

├─ max_tokens: 800→1200→1600→2000
├─ 动画复杂度：逐级递增
└─ 代码难度：从无到机器学习实现
```

### 第12页 · 个性化学习路径

**自适应学习闭环**

```
学习 → 测验 → 掌握度分析 → 推荐下一步
  ↑                                    ↓
  └────── 兴趣偏好更新 ←────────────────┘
```

**掌握度自动判定**：
- 正确率 ≥ 80%（≥3次测验） → "已掌握" → 推荐进阶内容
- 正确率 ≤ 40% → "需加强" → 推荐教学课 + 动画 + 练习

**兴趣偏好追踪**：
- 按主题统计交互频率
- Top兴趣主题优先推荐
- 侧边栏展示"感兴趣的话题"

### 第13页 · 游戏化激励系统

**让学习像游戏一样上瘾**

**XP经验系统**
- 答对 +10XP
- 答错 +2XP（鼓励尝试，不惩罚）
- 实时显示"+10XP"动画

**等级机制**
- 每100XP升1级
- 等级越高所需XP越多
- 升级时弹窗通知"🎉 升级！"

**连击追踪**
- 连续答对计数
- 答错归零
- 显示"最长连击"记录

**侧边栏可视化**：等级徽章 + XP进度条 + 当前连击

### 第15页 · 技术创新点

**1. Agent编排架构**
7个专职Agent + Orchestrator调度，业界主流的Agent协作范式在教育领域的创新应用

**2. Office文档零依赖生成**
纯Python构建PPTX/DOCX（直接操作XML+ZIP），无需python-pptx等第三方库

**3. 三层学段自适应**
Prompt注入 + 知识库分级 + 参数调整，实现真正的认知梯度适配

**4. 分学段动画生成**
同一概念的HTML5动画按学段调整复杂度——从可爱配色到学术标注

**5. 完整的学习闭环**
学习→测验→掌握分析→兴趣推荐→游戏化激励，形成自学正循环
