# 多模态K12人工智能通识课教学助手 — 实现计划

## 1. 项目概述

基于大语言模型的K12 AI通识课程对话智能体，支持4个学段自适应、
至少3种多模态交互、个性化学习路径。

## 2. 技术选型

| 层 | 技术 | 理由 |
|---|---|---|
| 前端 | Next.js 14 + TailwindCSS | 快速构建、生态成熟 |
| 后端 | Python FastAPI | 异步支持好、与AI库兼容 |
| LLM | OpenAI-compatible API | 灵活切换模型 |
| 向量库 | ChromaDB | 轻量、Python原生 |
| 多模态 | 各AI API + playwright动画 | 按功能选最优方案 |

## 5. 知识库设计

覆盖4个学段、5大AI主题：
1. AI基础概念（什么是AI、机器学习vs编程）
2. 数据与算法（数据的作用、排序、搜索）
3. 机器学习入门（分类、回归、神经网络直觉）
4. 计算机视觉（图像识别、人脸识别原理）
5. AI伦理与社会影响

每个主题按学段分层：小学低(绘本故事) / 小学高(游戏互动) / 初中(概念+实践) / 高中(数学原理+编程)

## 6. Agent 分工

| Agent | 职责 | 触发条件 |
|-------|------|----------|
| Orchestrator | 意图识别、路由分发、上下文管理 | 每次对话 |
| Dialogue | 自由问答、概念解释 | 默认 |
| Teaching | 结构化课程讲授、知识卡片生成 | "上课/学习/教我" |
| Animation | 生成可视化动画 | "动画/演示/可视化" |
| PictureBook | 生成绘本故事 | "绘本/故事" |
| Coding | 编程教学 + 沙箱执行 | "代码/编程/写" |
| Quiz | 生成练习题 + 批改 | "练习/测验/考考我" |

## 7. 数据流

```
用户选学段 → System Prompt注入年级上下文
用户发消息 → Orchestrator识别意图
  ├── 自由问答 → Dialogue Agent + RAG检索
  ├── 上课请求 → Teaching Agent → 生成知识卡片
  ├── 动画请求 → Animation Agent → 生成HTML5动画
  ├── 绘本请求 → PictureBook Agent → LLM故事 + 图片
  ├── 编程请求 → Coding Agent → 代码模板 + 沙箱
  └── 练习请求 → Quiz Agent → 出题 + 实时批改
→ 流式返回前端 → 渲染对应组件
→ 记录学习历史 → 更新用户模型
```

## 9. 运行方式

```bash
# 后端
cd backend && pip install -r requirements.txt && uvicorn main:app --reload

# 前端
cd frontend && npm install && npm run dev
```

需要配置 `.env` 文件设置 LLM API key。
